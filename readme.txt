D&D Gamemaster Tool
version 0.2

This console tool is a helper for the gamemaster.
Functions:
- creature configuration in creatures.ini
    - attributes
    - dices
    - amount of attacks
    - description
    - dynamic names for every creature

- console menue:
    a.) creating a group of enemies
    b.) throw initiative and attacks
    c.) input player throws to calculate damage and death.
    d.) calculates exp points when the whole group is defeated

run python3 main.py to start the console menue.
